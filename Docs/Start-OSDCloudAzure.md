# Start-OSDCloudAzure

## SYNOPSIS
Installs the OSDCloudAzure Module and starts OS Deployment from Azure Storage

## SYNTAX

```
Start-OSDCloudAzure [-Force] [<CommonParameters>]
```

## DESCRIPTION
Installs the OSDCloudAzure Module and starts OS Deployment from Azure Storage

## EXAMPLES

### EXAMPLE 1
```
Start-OSDCloudAzure
```

### EXAMPLE 2
```
Start-OSDCloudAzure -Force
```

## PARAMETERS
