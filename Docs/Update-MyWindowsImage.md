# Update-MyWindowsImage

## SYNOPSIS
Updates a mounted WIM

## SYNTAX

```
Update-MyWindowsImage [[-Path] <String[]>] [[-Update] <String>] [-BitsTransfer] [-Force] [<CommonParameters>]
```

## DESCRIPTION
Updates a mounted WIM files. 
Requires WSUSXML Catalog
