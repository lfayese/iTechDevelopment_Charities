# Get-WinREPartition

## SYNOPSIS
Returns the Partition containing Windows Recovery Environment WIM

## SYNTAX

```
Get-WinREPartition [<CommonParameters>]
```

## DESCRIPTION
Returns the Partition containing Windows Recovery Environment WIM
This function must be run in Windows
