# Save-WebFile

## SYNOPSIS
Downloads a file from the internet and returns a Get-Item Object

## SYNTAX

```
Save-WebFile [-SourceUrl] <String> [-DestinationName <String>] [-DestinationDirectory <String>] [-Overwrite]
 [-WebClient] [<CommonParameters>]
```

## DESCRIPTION
Downloads a file from the internet and returns a Get-Item Object
