# Get-OSDCloudTemplate

## SYNOPSIS
Returns the path to the current OSDCloud Template. 
This is typically $env:ProgramData\OSDCloud\Templates\Default

## SYNTAX

```
Get-OSDCloudTemplate [<CommonParameters>]
```

## DESCRIPTION
Returns the path to the current OSDCloud Template. 
This is typically $env:ProgramData\OSDCloud\Templates\Default
