# Invoke-Exe

## SYNOPSIS
Run External Command.

## SYNTAX

```
Invoke-Exe [-Executable] <String> [[-Arguments] <Object>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
This function calls an external command outside of the powershell script and logs the output.

## EXAMPLES

### EXAMPLE 1
```
Invoke-Exe dir c:\
```
