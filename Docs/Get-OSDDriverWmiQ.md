# Get-OSDDriverWmiQ

## SYNOPSIS
Returns a Computer Model WMI Query that can be used in Task Sequences

## SYNTAX

```
Get-OSDDriverWmiQ [[-InputObject] <Object[]>] [[-OSDGroup] <String>] [[-Result] <String>] [-ShowTextFile]
 [<CommonParameters>]
```

## DESCRIPTION
Returns a Computer Model WMI Query that can be used in Task Sequences
