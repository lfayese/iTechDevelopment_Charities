# Get-DellWinPE10DriverPack

## SYNOPSIS
Returns the URL of the latest Dell WinPE 10 Driver Pack

## SYNTAX

```
Get-DellWinPE10DriverPack [<CommonParameters>]
```

## DESCRIPTION
Returns the URL of the latest Dell WinPE 10 Driver Pack

## EXAMPLES

### EXAMPLE 1
```
Get-DellWinPE10DriverPack
```

### EXAMPLE 2
```
$DellWinPEDriverPack = Get-DellWinPE10DriverPack
```
